# Bài tập Tuần 14 - Cấu trúc dữ liệu Cây

## 1. Thông tin sinh viên
* Họ và tên: Nguyễn Công Đông
* MSSV: 20233318

## 2. Cấu trúc dự án
Bài tập được chia làm 2 Project tương ứng với 2 cách cài đặt cây nhị phân:
* **Project 1 (ArrayBased):** Cài đặt cây bằng cấu trúc lưu trữ tuần tự (mảng).
* **Project 2 (PointerBased):** Cài đặt cây bằng cấu trúc lưu trữ móc nối (con trỏ/link list).

## 3. Chức năng đã cài đặt
Cả 2 project đều bao gồm các hàm:
* Khởi tạo cây, kiểm tra cây rỗng.
* Thêm nút con trái, con phải.
* Duyệt cây theo 3 phương thức: Tiền tố (NLR), Trung tố (LNR), Hậu tố (LRN).
